What is this?

CSP Visualizer is an educational tool inspired by VisuAlgo and AI-Space that helps you understand how Constraint Satisfaction Problems are solved. Watch algorithms work step-by-step, explore different heuristics, and gain intuition about constraint propagation and backtracking search.

Algorithms Explained

Backtracking Search

The basic algorithm tries to assign values to variables one at a time, checking constraints after each assignment. If a constraint is violated, it backtracks and tries a different value.

Forward Checking (FC)

After each assignment, Forward Checking removes inconsistent values from the domains of unassigned neighbors. This detects failures earlier than basic backtracking.

AC-3 (Arc Consistency)

AC-3 is a constraint propagation algorithm that enforces arc consistency by iteratively removing values that have no supporting values in neighboring domains. It's more powerful than Forward Checking as it propagates constraints transitively.

Heuristics

  • MRV (Minimum Remaining Values) ‚Äî Choose the variable with the fewest legal values first. This fails fast on dead ends.
  • LCV (Least Constraining Value) ‚Äî Try the value that rules out the fewest choices for neighboring variables. This maximizes flexibility.

  "type": "graph",
  "description": "A custom CSP",

    {"name": "A", "domain": [1,2,3], "position": {"x": 100, "y": 100}}
  ],
  "constraints": [
